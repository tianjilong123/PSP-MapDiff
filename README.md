# PSP-MapDiff
We will provide the datasets and code of the article "PSP-MapDiff: Progressively semantic prior information-guided online map generation from remote sensing image via diffusion model"

### 11.14 updating

We have provided our datasets for public: https://pan.baidu.com/s/1XlPpA361WEJ3IGCC5veCaw?pwd=dbrg
